LIGHT GROUPS

A new type of Cycles render pass that contains only the lighting from
a subset of light sources.

Light Groups can be used in order to e.g. modify the color and/or
intensity of light sources in the compositor without re-rendering.

All Light Groups combined in the Compositor.
